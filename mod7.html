This is a demo developed with the Kha library, the ZUI sub-library, and the Haxe language. The Haxe language is isomorphic, meaning you can write the code once 'iso' and it can be compiled 'morphed' into other languages targeting different platforms and IDE's. In fact, I was able to build this demo for Linux, OSX, IOS, Windows, Android, Flash and HTML5 with only one set of code!

Much thanks to Lewis Lepton for his Kha tutorial series.
